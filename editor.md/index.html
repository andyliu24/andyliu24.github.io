最新版本： v1.1.6，更新于2015-03-02

Github下载

或者通过Bower安装：

bower install editor.md

依赖项目

Editor.md 的诞生依赖于以下开源项目：

  • CodeMirror
  • marked
  • jQuery
  • FontAwesome
  • github-markdown.css
  • KaTeX
  • Rephael.js
  • prettify.js
  • flowchart.js
  • sequence-diagram.js

开发构建使用到的项目及工具：

  • Brackets（推荐使用）
  • Sass/Scss
  • Gulp.js

开源协议

Editor.md 建立在众多优秀的开源组件基础之上，遵循和使用 MIT License 开源协议，无论个人还是公司，都可以免费自由使用。
